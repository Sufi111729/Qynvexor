## Keep bundles lean
Split routes, avoid large utility dependencies, and remove dead code early.

## Optimize rendering
Memoize heavy UI blocks and avoid unnecessary state updates in shared components.

## Build for mobile first
Test on slower networks and smaller devices to keep interactions reliable.
